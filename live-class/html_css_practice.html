블록(Block) 요소와 인라인(Inline) 요소

블록 요소(div, p, h1 등)는 한 줄을 모두 차지하며, 인라인 요소(span, a, strong 등)는 콘텐츠 크기만큼만 공간을 차지합니다.

박스 모델의 이해

모든 HTML 요소는 Margin, Border, Padding, Content로 구성된 사각형 박스입니다. `box-sizing: border-box;`를 사용하면 레이아웃 설계가 편리해집니다.
